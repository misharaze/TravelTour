 ЧудаТур — Туристический сайт

Современный адаптивный сайт туристического агентства с формой заявки, модальным окном, SEO-оптимизацией и серверной отправкой заявок.

 Функционал
- Адаптивная верстка
- Bootstrap 5
- Модальная форма заявки
- Отправка заявок на почту
- SEO-оптимизация
- Оптимизированные изображения
- Поддержка мобильных устройств

 Стек технологий
- HTML5
- SCSS / CSS3
- Bootstrap 5
- JavaScript
- Node.js (сервер для отправки заявок)
- Git / GitHub

 Контактная форма
Форма позволяет отправлять заявки клиентов прямо на почту администратора.

 Деплой
Проект подготовлен к деплою на Vercel.
